## Firefox / Librewolf Contents

So the last element, addons I will always use:

* [Clear URLs](https://addons.mozilla.org/en-US/firefox/addon/clearurls/)

* [DuckDuckGo Privacy Essentials](https://addons.mozilla.org/en-US/firefox/addon/duckduckgo-for-firefox/)
* [Enhancer For Youtube](https://addons.mozilla.org/en-US/firefox/addon/enhancer-for-youtube/)
* [I still dont care about cookies](https://addons.mozilla.org/en-US/firefox/addon/istilldontcareaboutcookies/)
